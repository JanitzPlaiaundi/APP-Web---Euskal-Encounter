PROYECTO DAW EUSKAL ENCOUNTER

El objetivo de este proyecto es que en un tiempo determinado, tengamos una pagina web al completo dedicada a la empresa que se supone que nos ha mandado el encargo de crearla.

Los contenidos de este proyecto seran HTML-s de paginas y demas archivos complementarios para su mejor visualizacion, interactividad y intuitividad.
Tambien nos permitiria conectarla con una aplicacion de escritorio programada en java, pendiente de hacer.
